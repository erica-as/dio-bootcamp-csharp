PROPRIEDADES: 

Propriedades permitem que uma classe exponha uma forma pública de obter e definir valores, sem a
necessidade de implementação ou verificação obrigatória. 

Atalho para criar uma propriedade => 
1. prop + tab
2. Os nomes das propriedades sempre sempre devem começar com letra maiúscula seguindo a nomeclatura CamelCase.

MÉTODOS: 

Os métodos GET e SET são usados para obter ou definir os valores de uma propriedade. Caso necessário, é possível criar 
validações para definição de um SET. 

Para criar validações => 
1. Body expressions: (=>)
2. Operadores condicionais 

TIPOS DE CLASSES, PROPRIEDADES E MÉTODOS:

Public => Ele pode ser usado por qualquer classe pode acessar o instancuiar essa propriedade.
Private => Ele só pode ser acessado dentro da classe.  

DIFERENÇAS ENTRE CAMPO E PROPRIEDADES: 

Necessariamente, uma propriedade tem que ter um get ou set, o campo não é necessário, apenas a definição do tipo do campo e o nome.
